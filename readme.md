# Backend - Social Typescript

Projeto construíndo com intuito de treinar Typescript no front-end juntamente com ReactJS. Essa API vai contem funções de Auth e de Posts

## Tecnologia usadas: 

1- NodeJS;
2- Express;
3- MongoDB;

## Features

[x] Criação de login
[x] Criação de post
[x] Edição de post
[x] Thumbs up nos post
